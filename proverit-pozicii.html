Инструмент имеет возможность бесплатно проверить позиции сайта в поисковых системах Яндекс и Google по заданным ключевым словам. Все снятые данные запоминаются и хранятся в истории. Для проверки введите в текстовое поле запросы (каждый с новой строки) и в соответствующее поле название домена. Бесплатная проверка позиций осуществляется на глубину 200 результатов для Yandex и 100 для Google (+100 платная), максимальное количество проверяемых запросов в сутки 25 для каждой ПС, в выходные 70, в день апдейта 20. Определение позиций сайта работает в последовательном порядке по 1 запросу. По умолчанию позиции определяются для региона "Москва", введите соответствующий код для проверки позиций в вашем регионе. Код вашего региона в Яндекс поисковике можно узнать в адресной строке при отправке поискового запроса: http://yandex.ru/yandsearch?text=zapros&lr=213 - в данном случае код 213 (Москва). Определение позиции сайта в Яндексе идет через реальную выдачу. Если навести курсор мыши на результат проверки, вы увидите релевантную страницу в поисковых системах.
